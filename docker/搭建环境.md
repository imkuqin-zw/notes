## 搭建的服务
* apisix
    * apisix-server
    * apisix-dashboard
* polaris 
    * polaris-server
    * polaris-limiter
    * polaris-console
* monitor 监控相关组件
    * etcd
    * grafana
    * prometheus
* database 数据库相关组件
    * mysql

### 启动顺序
1. monitor
2. database
3. polaris
4. apisix


### 设置apisix代理
1. 修改本地hosts
    ```json
    127.0.0.1 console.polaris.local
    127.0.0.1 dashboard.apisix.local
    127.0.0.1 grafana.local
    ```
 2. 设置apisix路由
    * console-polaris
      ```json
        {
          "uri": "/*",
          "name": "console-polaris",
          "methods": [
            "GET",
            "POST",
            "PUT",
            "DELETE",
            "PATCH",
            "HEAD",
            "OPTIONS",
            "CONNECT",
            "TRACE",
            "PURGE"
          ],
          "host": "console.polaris.local",
          "upstream": {
          "nodes": [
            {
            "host": "polaris-console",
            "port": 8080,
            "weight": 1
            }
          ],
          "timeout": {
            "connect": 6,
            "send": 6,
            "read": 6
          },
          "type": "roundrobin",
          "scheme": "http",
          "pass_host": "pass",
          "keepalive_pool": {
            "idle_timeout": 60,
            "requests": 1000,
            "size": 320
            }
          },
          "status": 1
        }
        ```
